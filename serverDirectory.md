

# 이 글은 서버 디렉토리에 대한 설명을 작성한 문서이다.

##서버 디렉토리 구성


### 계층 1단계 설명 

- server
  - /asset : 블로그 콘텐츠, 데이터 저장 
  - /dist :  블로그페이지 구현시 필요한 root script(JS, CSS등)
  - /img : 블로그전용 이미지폴더
  - /font :  블로그 전용 폰트폴더
  - /sass : for sass complie
  - index.html : 블로그 메인 페이지 마크업


### 계층 2단계 설명
- server

    - /contents : 블로그 콘텐츠 저장 폴더

  - /dist
